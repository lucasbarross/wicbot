* Documentation (WIP)

### Commands:
* _%dc jogar (play)_ - **Starts the game**
* _%dc sair (quit)_ - **Finishes a game**
* _%dc ajuda (help)_ - **Shows the command dialog**
* _%dc skip (pular)_ - **Skips a champion**
